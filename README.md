Hola! los pasos para montar la web son:

1.-

### `npm install`

2.- cuando termine de instalas todas las dependencias poner:

### `npm start`

3.- en otra terminal que este apuntando a la carpeta del proyecto poner: json-server db.json --port 4000

### `json-server db.json --port 4000`

esto arrancara el fake server que simulara una db con apis
